The WolfBin

Pastebin minimalista, cifrado y de código abierto

¿Qué es The WolfBin?

The WolfBin es un pastebin online minimalista y de código abierto donde el servidor no tiene conocimiento alguno de los datos almacenados. Los datos se cifran y descifran en el navegador mediante AES de 256 bits en Galois Counter Mode.

¿Qué ofrece?

  • Eliminación de contenido bajo demanda
  • Sistema similar a Pastebin para almacenar documentos de texto, ejemplos de código, etc.
  • Cifrado de datos enviados al servidor
  • Protección con contraseña para leer el archivo pegado
  • Tiempos de expiración: "para siempre" o "quemar después de leer"
  • Soporte para Markdown y vista previa de HTML
  • Resaltado de sintaxis para código fuente con prettify.js y 4 temas
  • Soporte para carga de archivos, imágenes, multimedia y vista previa de PDF
  • Plantillas personalizables: Bootstrap5, Bootstrap CSS y Darkstrap
  • Detección automática de idioma y sistema de traducción
  • Generación de código QR para compartir URLs fácilmente

¿Qué NO ofrece?

La clave utilizada para cifrar el archivo pegado forma parte de la URL. Si publicas la URL de un archivo pegado sin contraseña, cualquiera podrá leerla. Usa una contraseña segura y compártela de forma privada y cifrada de extremo a extremo si quieres que tu archivo pegado permanezca privado.
